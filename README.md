## Stack técnico

- **Three.js r0.160.0** — Energy Cube 3×3×3 (MeshPhysicalMaterial, EdgesGeometry, ACESFilmic)
- **Canvas 2D** — Infinite perspective grid + plexus network (partículas + líneas)
- **CSS3** — Animaciones, glassmorphism, gradientes de marca
- **Google Fonts** — Abril Fatface + Inter

## Uso

Cada template es un archivo HTML standalone. Abrir directamente en navegador. Los campos editables están marcados con `★ EDITABLE` en los comentarios.
